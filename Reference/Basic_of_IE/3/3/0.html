生产率概述


      生产率是工业工程中的一个重要概念，提高生产率是工业工程的出发点和奋斗目标，工业工程从产生之日起就是为了追求更高的效率和生产率。

      随着科学发展和技术进步，工业工程得到迅速发展，并越来越广泛地应用于各种生产领域，不断提高生产率水平，促进经济增长。而经济发展水平的提高又对生产率增长提出了新的更高的目标，促使企业应用工业工程去作进一步的改进。于是，提高生产率与应用和发展工业工程之间形成了相互促进的关系。
